Mind Garden is an iOS app built as a Swift Package Manager-based application, designed for adult children of people with Alzheimer’s disease, or anyone who wants to pay closer attention to their parents’ cognitive health. Through three gentle interactive experiences and a locally generated daily Mind Report, it helps families notice changes in attention, short-term memory, and cognitive flexibility in a more intuitive and less stressful way. Rather than giving medical conclusions, Mind Garden aims to offer a softer starting point for understanding, conversation, and earlier care.
 Platform

  - Swift Package Manager app
  - Recommended platform: iPadOS 26 and above
  - Recommended orientation: Landscape
  - Minimum supported version: iPadOS 18.0
